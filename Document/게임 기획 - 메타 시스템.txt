*표시는 미확정이며 고민이 좀 더 필요함

캐릭터 시스템
- 고유 스킬 1개: 쿨타임이 있는 액티브 스킬
- 외형: 차별화
- 성능: HP/MP/쉴드 등 차별화
- *캐릭터별 영구적으로 능력치 강화

장비 시스템
- 무기: 근접/원거리/마법/레이저 등
- 방어구: HP/쉴드/방어력/회피 등
- 장신구: 패시브 효과, 파밍/스킬 시너지
- 획득 경로
   → 방 클리어 시 랜덤 보상 상자
   → 상점방 구매
   → 이벤트 보상
- 장비 티어
   → 일반 → 레어 → 에픽 → 전설
   → 티어가 오를수록 능력치 + 전용 효과 추가

방 탐험에 의한 보상 구조
- 방 클리어: 랜덤 보상 상자 n개 지급 (일반 1개, 엘리트 2개)
   → 무기
   → 방어구
   → 장신구
   → 골드: 체력 소모, 저주, 아무 것도 없음 등
   → 꽝
- 상자 드랍률 조정
   → 초반: 낮은 티어 위주 + 소량 골드
   → 중반: 레어 티어 확률 증가
   → 후반: 에픽/전설 확률 증가, 꽝도 늘려 긴장감 유지

파밍/빌드 메이킹 루프
1. 로비 → 던전 입장
2. 방 탐험 → 전투 클리어
3. 상자 오픈 → 무기/방어구/장신구/골드/꽝 랜덤
4. 빌드 점점 강화
5. 보스방 경고 → 들어갈지 말지 선택
6. 보스 처치 → 다음 층 진행
7. 총 3층까지 진행

영구 보상 & BM 연계
- 플레이 종료 후 영구 획득: 정수/영혼
- 정수: 새 콘텐츠 해금
   → 캐릭터 해금
   → 장비 해금
   → *로비에서 기능 해금 (예: 정수 500 → 대장간 해금 → 무기 합성 가능)
- 영혼: 해금된 콘텐츠 강화
   → 특정 무기/방어구/장신구의 영구 능력치 강화
   → 특정 캐릭터 영구 능력치 강화
- BM 구조
   → 유물 뽑기 (포션 개수 증가, 쉴드량 증가, 이동속도 상승 등)
   → 광고 (부활 1회 등)

이벤트 방 종류 (층별 난이도에 맞추기)
- 자원 교환형
  → 체력 소모 ↔ 장비
  → 골드 ↔ 포션
- 도박형
  → 랜덤 상자: 장비/골드/정수/영혼/꽝
  → 신비한 상인 (골드): 장비/장비 강화/장비 티어 상승/정수/영혼/꽝
  → 저주 제단: 디버프(최대 체력 감소, 이동속도 감소 등) 부여 대신 장비/정수/영혼/ 지급
  → 버프 제단: 랜덤 버프/디버프 부여 (이동속도 증가/최대 HP 감소 등)
- 강화형
  → 무기 개조소: 현재 장비 중 하나 강화
  → 스킬 훈련소: 캐릭터 고유 스킬 효과 업그레이드
  → 장신구 각인: 장신구에 추가 옵션(치명타, 흡혈 등) 부여
- 회복/지원형
  → 치유의 샘물: 체력/쉴드/포션 충전
  → 석상: 버프 부여
